01. open, alert, confirm, prompt 메소드

    자바스크립트 언어는 브라우저상에서 동작하는 언어다.
    자바스크립트가 web 상에서 동작할 때 사용할 수 있는 기능이 있다.
    web 상에서 가장 최상위 객체는 window 객체이다.

    window
        - DOM (document 등등)
        - BOM (location, navigator 등등)
        - JavaScript (Array, Function, Object 등등)

    최상단의 window 객체는 브라우저 창을 대변하고 브라우저를 제어할 수 있는 메소드를 제공한다.
    window 는 전역 객체이기 때문에 생략이 가능하다.
        ex] window.alert() => alert()

open

    window.open(url, name, params) 메소드로 새로운 창을 열 수 있다.

alert

    window.alert(message) 메소드는 확인 버튼을 가지며 
    메세지를 지정할 수 있는 경고 상자를 띄워준다.

confirm

    window.confirm(message) 메소드는 확인과 취소 두 버튼을 가지며 메세지를 지정할 수 있게 된다.
    반환값이 존재하며 확인을 누르면 true, 취소 or ESC 를 누르면 falsc 를 반환해준다.
