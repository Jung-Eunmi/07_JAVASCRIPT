01. 이벤트 핸들러 어트리뷰트 방식

이벤트 핸들러 등록

    이벤트 핸들러
     - 이벤트가 발생했을 때 브라우저에 호출을 위임한 함수
     - 이벤트가 발생 했을 때 브라우저에게 이벤트 호출을 위임하는 것을 이벤트 핸들러 등록이라고 하며,
       이벤트 핸들러를 등록하는 방법은 3가지가 있다.

이벤트 핸들러 어트리뷰트 방식

    HTML 이벤트 핸들러 어트리뷰트(on 접두사 + 이벤트 타입) 값으로
    함수 호출문을 할당하여 이벤트 핸들러를 등록하는 방식이다.
    주의점은 함수 참조가 아닌, 함수 호출문을 할당하는 것이다.
    
    어트리뷰트 방식으로 이벤트 핸들러를 등록하게 되면 
    HTML/CSS 요소뿐아니라 JavaScript 언어도 추가된다.
    따라서 JavaScript 언어를 모르는 기획자나 퍼블리셔들은 혼란에 빠질 수 있다.

    HTML 은 정적인 요소, JavaScript 는 정적인 요소를 동적으로 바꿀 수 있는 요소이다.
    그러므로 두개의 요소는 따로 분리해두는 것을 추천하지만,
    나중에 배울 SPA 리액트에서는 이벤트 핸들러를 어트리뷰트 방식으로 처리하기 때문에 알아두는것이 좋다.
